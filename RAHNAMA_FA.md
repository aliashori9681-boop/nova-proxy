# نوا پروکسی — تغییرات و چک‌لیست تست

این چک‌لیست کمک می‌کند بعد از استقرار، همه‌ی امکانات را روی دامنه‌ی خودت (`<your-domain>`) تست کنی.

## ✅ چک‌لیست تست (روی `<your-domain>`)

**اشتراک و فرمت‌ها**
- [ ] لینک‌های `‎/sub/mihomo.yaml`، `‎/sub/base64.txt`، `‎/sub/singbox.json` در **Clash Party / Mi Clash / FLClash / Hiddify / v2rayNG** بدون خطا وارد شوند.
- [ ] دیگر خطای «unable to determine config format» در Hiddify نباشد.

**شبکه و DNS**
- [ ] آدرس `‎/dns-query?name=google.com&type=A` جواب JSON بدهد؛ `‎/dns-query/@هرچیزی` هم کار کند.
- [ ] با **«سایت‌های ایرانی مستقیم»** روشن، یک سایت `.ir` (مثل ipmyp.ir) را باز کن → باید **IP واقعی خودت** را نشان دهد، نه IP پروکسی.
- [ ] DNS کانفیگ‌های Clash حالا ایرانی/انتخابی توست، نه چینی.

**WARP**
- [ ] صفحه‌ی شبکه → WARP → **ثبت** → «حساب WARP ثبت شد». بعد یک کلید **WARP+** → **اعمال**.
- [ ] اگر ثبت با خطای X25519 داد، `compatibility_date` را در `wrangler.jsonc` جدیدتر کن.

**داشبورد و تلگرام**
- [ ] کلید **اعلان تلگرام** روشن/خاموش شود؛ Account ID / توکن / Bot / Chat بعد از جابه‌جایی صفحه خالی نشوند.
- [ ] مصرف با کاما نمایش داده شود (مثل `۲۴٬۳۴۵ / ۱۰۰٬۰۰۰`) و نوار رنگی باشد.
- [ ] ربات را ذخیره کن → **`/start`** جواب بدهد (وب‌هوک خودکار نصب می‌شود).

**امنیت (جدید)**
- [ ] تنظیمات → کارت **امنیت** → **تغییر رمز**: رمز فعلی + رمز جدید → ذخیره. بعد از تغییر، **کانفیگ‌های فعلی کاربران باید همچنان کار کنند** (شناسهٔ پروکسی ثابت می‌ماند).
- [ ] **احراز هویت دو مرحله‌ای (2FA)**: «فعال‌سازی» را بزن → QR را با **Google Authenticator / Authy / Keeper** اسکن کن → کد ۶ رقمی را وارد و تأیید کن.
- [ ] خروج و ورود دوباره → بعد از رمز، یک **کد ۶ رقمی** خواسته شود.
- [ ] (نکتهٔ بازیابی) اگر برنامهٔ احراز هویت را از دست دادی، کلید `admin_2fa.json` را از KV ورکر پاک کن تا دوباره فقط رمز خواسته شود.

**قوانین**
- [ ] **قوانین آماده** (عبور چین/روسیه، مسدودسازی QUIC/بدافزار/…) و **قوانین دلخواه** اعمال شوند.
- [ ] **اعلان سرویس رایگان**: در هر کانفیگ یک نود برچسب‌دار `🎁 Free service — novaproxy.online` باشد و **قابل خاموش‌کردن یا تغییر نباشد**.

**تنظیمات پیشرفته (اختیاری)**
- [ ] TLS fragment → **سفارشی** با طول/فاصله/بسته‌ی دلخواه.
- [ ] Custom CDN (Host/SNI) و ECH (SNI/DoH) ذخیره و اعمال شوند.

## 🆕 چه چیزهایی اضافه/اصلاح شد

**روش بهبودیافته‌ی WhiteDNS**
- لینک‌های عمومی بدون توکن: `‎/sub/mihomo.yaml`، `‎/sub/base64.txt`، `‎/sub/singbox.json`.
- لنگرگاه دائمی روی گیت‌هاب (شاخه‌ی `sub`) که با فیلتر دامنه از بین نمی‌رود.
- استخر دامنه‌ی خوددرمان‌گر + اعلان خودکار تلگرام هنگام تغییر دامنه‌ها.
- بلِند: افزودن نودهای غیر-کلودفلر به‌عنوان جایگزین (`FALLBACK_SUB`).

**رفع باگ‌ها**
- DoH: مسیر `‎/dns-query/@توکن` و حالت JSON (`?name=`) درست شد.
- نشت IP در سایت‌های ایرانی برطرف شد (`DOMAIN-SUFFIX,ir,DIRECT`؛ حذف rule-setهای خراب).
- فرمت Hiddify/sing-box/Clash درست شد (`?target=`).
- تلگرام: نصب خودکار وب‌هوک؛ کلید فعال‌سازی درست شد؛ فیلدها دیگر خالی نمی‌شوند.
- DNS کانفیگ از حالت چینی به **ایرانی/انتخابی** تغییر کرد.

**قابلیت‌های جدید**
- WARP: ثبت حساب + لایسنس WARP+ + افزودن WARP به کانفیگ.
- ژنراتور WARP/WireGuard و NekoRay (`‎/warp`).
- قوانین آماده (انتخاب کشور) + قوانین دلخواه روی Clash.
- پیشرفته: Fragment سفارشی، Custom CDN، منابع ECH، نویز Warp.
- جابه‌جایی NAT64 (`NAT64`)، آمار ترافیک (روز/ماه/سال)، شناسه‌ی نمونه روی داشبورد.
- اعلان سرویس رایگان: **همیشه روشن و ثابت** (ضد فروشنده).
- صفحه‌ی **مدیریت** + قلاب‌های سرور مرکزی (heartbeat + اعلان همگانی) — سرورش روی `nova-ip-bot` ساخته می‌شود.
- **نصب یک‌کلیکی**: دکمه‌ی Deploy to Cloudflare + ویزارد نصب (`/install`) که داشبورد و KV را خودکار می‌سازد؛ فقط یک رمز می‌سازی.
- **امنیت**: تغییر رمز از داخل پنل (بدون شکستن کانفیگ کاربران) + **احراز هویت دو مرحله‌ای (2FA/TOTP)** سازگار با Google Authenticator/Authy/Keeper.

**رابط کاربری**
- یک دکمه‌ی ذخیره در هر صفحه، دکمه‌ها هم‌اندازه، راهنمای دوزبانه برای همه‌ی بخش‌ها.

## ⛔ چیزهایی که روی ورکر کلودفلر ممکن نیست
- صفحه‌ی ریدایرکت برای سایت‌های بزرگسال روی HTTPS (به‌خاطر TLS) — مسدودسازی کار می‌کند ولی ریدایرکت نه.
- Hysteria2 / TUIC / WireGuard به‌عنوان ورودی ورکر (ورکر UDP ورودی ندارد) — WARP فقط چون به سرویس WARP کلودفلر وصل می‌شود کار می‌کند.
- لینک با حجم/انقضای واقعی (کانفیگ‌های عمومی بعد از مرگ لینک هم کار می‌کنند).

اگر جایی خطا دیدی، در GitHub یک issue باز کن یا در تلگرام [@irnova_proxy](https://t.me/irnova_proxy) خبر بده 🙏
